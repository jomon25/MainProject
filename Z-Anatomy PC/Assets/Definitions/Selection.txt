

SELECTION


Press 'A' 
    to select all the visible objects
    
Press 'A;A' or 'Alt+A' 
    to deselect all

Press Left Mouse Button (LMB) 
    to select an object
    
Press 'Shift+LMB' 
    to Select several objects
    
Clic and drag 'LMB' 
    to select several objects with box-select
    
Press 'C'
    to enter/exit the brush selection mode
    Scroll the Mousewheel to adjust the brush size
    

Press 'RMB' 
    to Select one of the collections of the active object

Press 'LMB' twice (fast) 
    to extend the selection to the object's children

SELECCIÓN


Presione 'A'
para seleccionar todos los objetos visibles
 
Presiona 'A;A' o 'Alt+A'
para deseleccionar todo

Pulse el botón izquierdo del ratón (LMB)
para seleccionar un objeto
 
Presiona 'Shift+LMB'
Seleccionar varios objetos
 
Haga clic y arrastre 'LMB'
seleccionar varios objetos con box-select
 
Presione 'C'
para entrar/salir del modo de selección de pincel
Desplace la rueda del ratón para ajustar el tamaño del pincel
 

Presione 'RMB'
Seleccionar una de las colecciones del objeto activo

Presione 'LMB' dos veces (rápido)
para extender la selección a los hijos del objeto

SELEÇÃO


Pressione 'A'
para selecionar todos os objetos visíveis
 
Pressione 'A;A' ou 'Alt+A'
para desmarcar todos

Pressione o botão esquerdo do mouse (LMB)
para selecionar um objeto
 
Pressione 'Shift+LMB'
para selecionar vários objetos
 
Clique e arraste 'LMB'
para selecionar vários objetos com box-select
 
Pressione 'C'
para entrar/sair do modo de seleção de pincel
Role a roda do mouse para ajustar o tamanho do pincel
 

Pressione 'RMB'
selecionar uma das coleções do objeto ativo

Pressione 'LMB' duas vezes (rápido)
para estender a seleção aos filhos do objeto

SÉLECTION


Appuyez sur 'A'
pour sélectionner tous les objets visibles
 
Appuyez sur 'A;A' ou 'Alt+A'
pour tout désélectionner

Appuyez sur le bouton gauche de la souris (LMB)
sélectionner un objet
 
Appuyez sur 'Maj+LMB'
sélectionner plusieurs objets
 
Cliquez et faites glisser 'LMB'
pour sélectionner plusieurs objets avec box-select
 
Appuyez sur 'C'
pour entrer/sortir du mode de sélection de brosse
Faites défiler la molette de la souris pour ajuster la taille du pinceau
 

Appuyez sur 'RMB'
pour sélectionner une des collections de l'objet actif

Appuyez deux fois sur 'LMB' (rapidement)
pour étendre la sélection aux enfants de l'objet
