◆ SelectAllChildren()

void SelectedObjectsManagement.SelectAllChildren ( Transform  parent,
bool  lastIteration = true,
List< GameObject >  shown = null 
)

Recursively selects all the children of a parent Transform, excluding objects with Line or Label components and objects with the tag "Insertions". Selected objects are added to the list of selectedObjects and their visibility is set to true. If the child object is not active in the hierarchy, it is set active and its parents are also set active recursively.

Parameters
parentThe parent Transform of the children to select.
lastIterationA bool that specifies if this is the last iteration of the recursive function. Default is true.
shownA List of GameObjects that specifies which objects are to be shown. Default is null.

summary> Handles click event to select the parent of the current selected object, and show its children objects in the scene. /summary>
