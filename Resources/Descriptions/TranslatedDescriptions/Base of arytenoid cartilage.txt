

BASE OF ARYTENOID CARTILAGE


The base of each cartilage is broad, and on it is a concave smooth surface, for articulation with the cricoid cartilage.

    -Its lateral angle is called the muscular process.
    -Its anterior angle is called the vocal process.

BASE DEL CARTÍLAGO ARITENOIDEO


La base de cada cartílago es ancha, y sobre ella hay una superficie lisa cóncava, para la articulación con el cartílago cricoides.

-Su ángulo lateral se denomina apófisis muscular.
-Su ángulo anterior se denomina apófisis vocal.

BASE DA CARTILAGEM ARITENÓIDE

BASE DU CARTILAGE ARYTÉNOÏDE


La base de chaque cartilage est large, et sur elle se trouve une surface lisse concave, pour l'articulation avec le cartilage cricoïde.

-Son angle latéral s'appelle le processus musculaire.
-Son angle antérieur s'appelle le processus vocal.
